MyDynamicNFT

Inherits: Initializable, ERC1155Upgradeable, AccessControlUpgradeable, ERC1155SupplyUpgradeable, UUPSUpgradeable

Author: Yuren Jin

This contract implements an ERC1155 token with dynamic evolution capabilities. Token evolution can change the token's attributes or utilities, representing user loyalty or achievements.
